Creating the header and nav

The header is standard, with a 'meta' tag to set the viewport for responsive design. The nav is made up of three links. One will be an image that will always link to the home page, an about me link and a projects link. These can be further added to or drop down menus can be created for them. I put all of the elements in the body tag and make regular use of relative links addresses throughout the project.

The body and image

The main chunk of the website is wrapped in a 'div' container class 'article'. This makes it easy for me to position the different pieces of the text within this container. A '.svg' image is used for the main graphic for its ability to resize without losing quality. Both of these containers will be easy to position above, below, or beside eachother.

The footer

The footer is standard as well. Nothing special. Each 'p' tag will be easy to float and wrap as the sizes change. I end the 'body' tag, then 'html' tag at the very end. With these two I can easily change the text color, styles, background colors, and other attributes of the entire web page. MY favorite part about this project is how simple and easy the tags are to organize. Pieces like the header, nav, and footer will be copy/pasted across every page on the site. This makes it easy and quick to update.
