Working With Files

Creating Directories

Initial File Setup

The method starts by checking if the "months" directory was made. If it was not then it will create the directory. Afterwards, the method checks if the file "mod.csv" was created. The file "mod" stands for "modified". This file will be used to assist in the deletion and modification of entries. If it was created then the method prints an empty string and if not then the method creates the file for future use.

Creating Files

Creating Month Files

Only months that are entered by the user will have a file associated with them. This method pulls the user entry and checks if a month was created. If it was not then the method creates the month file. Otherwise, it prints an empty string.

Listing Files

This method uses a built-in python library called OS that has a function that creates a list of directories in a file (listdir). By calling listdir(path), you can create a list of the file names and then iterate through them as menu items. In this case, the only month files that were created by user entry was January, February, and March.
