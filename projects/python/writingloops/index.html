Simple While Loop

This is an example of how I write my while loops. I start by setting a variable (stayInLoop = True) then nest further instructions within. This loop will only exit after returning a value. In this case the value being returned is (dayInput).

Nested While Loop

Here I have a while loop (stayInLoop2) inside another while loop. The first while loop appends items to a list (itemList) then the second while loop prompts the user to enter new data. The second while loop will only exit when the user enters 1 or 3.

Simple For Loop

For loops in my program are used to iterate through lists and files. This particular for loop pulls each items from a list of directories (monthFiles), capitalizes the first letter (.title()), then removes the .csv from the file name (.replace()) and appends it to a dictionary (monthDict). The dictionary will be used later to pull the value of the month using an integer set by x (monthDict[x+1] = month).

Nesting in for loop

I used a nested if statement in this for loop to compare a dictionary of months in order (monthOrder) then write the month to a new dictionary (monthKey) if it is a valid month. I added an else statememt at the end to continue iterating through the months if something in the list of directories was not valid. This will allow other directories to be added to "/months" without messing up the intended value-key pairs.
