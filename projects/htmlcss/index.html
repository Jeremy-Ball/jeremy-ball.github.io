Creating my portfolio

This website was created by replicating an example created by Victor Eke to showcase a headless CMS known as Sanity. I opted to write the whole thing from scratch to practice writing in HTML and CSS. Understanding how different types of code work together is an important skill for any programmer or network security specialist. This website is a constant work in progress and I will be adding content as I broaden my skills in programming and security. This website was made using mobile-first approach.
