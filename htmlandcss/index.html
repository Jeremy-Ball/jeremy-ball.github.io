Building a website using HTML and CSS

Step 1

The first thing I do is create a background color for the HTML document and a background color for the body of the website. This helps me visualize the contents of the website. Since I'm using Flexbox, I create a border around the container (red) then a black border around each item I will need. The basic layout of my design is simple with a header, a footer, and the main content in between. By setting it all up in the beginning I can play around with margins and prepare the website for responsive design.

Visualizing Flexbox

The green lines represent the cross axis that the items are lined up on. This orientation is set by using "flex-flow: row wrap;". This will set the items side by side until they cannot fit in the row any longer. By manipulating the width of each item we can create rows with 2 items and rows with 1.

Step 2

Here I add the appropriate text in each box. My header will stay the same across the rest of my pages so I finish that first. After, I will set the margins, text sizes, padding, etc. It's still in a very rough stage but I can spend a while at this step deciding how I want my header to look. I might want to add a banner background, a logo, or even work on my navigation links and have them complete before I work on the rest of the page.

Step 3

When I'm satisfied that the header and footer look the way I want I will remove the borders. The borders themselves will throw off the size of the flex items. Now I can resize them exactly the way I want and leave them while I work on other parts of the site. The header and footer can still be edited later, but now I can quickly throw up other pages on my website without having to worry about those two parts.

Step 4

I decided to work on the article first. I set the headings, paragraphs, and quotation with appropriate margins and padding. These can be edited later but now I have a good idea of what the end result of my site will be. After I pick a picture I can remove borders and touch up the margins to make it fit neatly together.

Step 5

This is the real "bread & butter" of Flexbox. You can manipulate the flex items to be centered across an axis. I changed the height of the article to be 30% and the width at 49.8% (as close as I can get with borders). When I center the article using align-self it moves it so the axis runs across the middle of the item. I used a green line to show the cross-axis that the items sit on. This is very useful for responsive design as the items will always sit in this orientation until I change them with a media query.

Step 6

The completed layout will have a radient background. The best part about this is that you can change the colors very quickly with just one line of code. I'll play with the margins and font-sizes, as well as font styles. But for the most part the layout is complete. We can begin with arguably the most important aspect of web development: Responsive Web Design.
