The program

I thought of this program when I needed to save money on eating out since I was doing it almost every day. I wanted to track how much money I was spending on specific grocery items (for work lunches) and compare it to how much I was spending eating out. The program was a lot of fun as it incorporated file reading/writing, lists, while/for loops, and input validation. Many hours went in to learning and practicing every aspect of these pieces.
